## 正文（V2EX/掘金版）

### 🎯 介绍

大家好，推荐一个我最近开发的开发者工具：**DevPrompt - 开发者 Prompt 模板库**。

这是一个收集了 50+ 高质量 AI Prompt 模板的网站，帮助开发者更高效地使用 ChatGPT/Claude 写代码。

### ✨ 核心功能

- ✅ **50+ 模板**：覆盖代码解释、Bug 修复、测试、文档、DevOps 等场景
- ✅ **分类清晰**：按使用场景分类，支持搜索
- ✅ **一键复制**：点击即可复制 Prompt
- ✅ **完全免费**：MIT 开源协议
- ✅ **响应式**：支持移动端使用

### 📦 技术栈

- Next.js 14 + Tailwind CSS + TypeScript
- 完全开源，欢迎贡献

### 🚀 使用方法

1. 访问网站（或本地运行）
2. 选择分类或搜索关键词
3. 点击模板查看详情
4. 一键复制 Prompt
5. 在 ChatGPT/Claude 中使用

### 🎨 适用场景

- 🔍 代码解释器 - 让 AI 详细解释代码逻辑
- 🐛 Bug 调试助手 - 快速定位和修复 Bug
- 🧪 单元测试生成 - 自动生成测试用例
- 📝 API 文档生成 - 根据代码生成文档
- 🐳 Docker 配置 - 生成最佳实践 Dockerfile
- ⚛️ React/Vue 组件 - 快速生成组件代码
- 📐 正则表达式 - 根据需求生成 Regex
- ☁️ AWS 基础设施 - Terraform 配置
