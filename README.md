## 🎯 关于卡皮巴拉

🦫 **卡皮巴拉**（Capibara）是一种可爱的水豚，以温柔、放松著称。

Kapibala 的使命：
- 像卡皮巴拉一样**温暖**（用户至上）
- 像卡皮巴拉一样**悠闲**（使用简单）
- 像卡皮巴拉一样**治愈**（让编码更快乐）

## 🚀 快速开始

### 在线访问（即将上线）
https://devprompt.vercel.app

### 本地运行

```bash
# 克隆项目
git clone https://github.com/yourusername/kapibala.git
cd kapibala

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000
