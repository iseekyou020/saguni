## 正文

大家好，推荐一个我最近开发的开发者工具：**Kapibala - AI Prompt 模板库**。

🦫 **卡皮巴拉**（Capibara）是一种可爱的水豚，以温柔、放松著称。希望这个工具能让你写代码更快乐！

### ✨ 核心功能

- ✅ **45+ 高质量模板**：代码解释、Bug 修复、测试、文档、DevOps 全覆盖
- ✅ **分类清晰**：21 个分类，支持搜索
- ✅ **一键复制**：点击即可复制 Prompt
- ✅ **完全免费**：MIT 开源
- ✅ **中文友好**：界面和说明都是中文

### 📦 适用场景

- 🔍 代码解释器 - 让 AI 详细解释代码逻辑
- 🐛 Bug 调试助手 - 快速定位和修复 Bug
- 🧪 单元测试生成 - 自动生成测试用例
- 📝 API 文档生成 - 根据代码生成文档
- 🐳 Docker 配置 - 生成最佳实践 Dockerfile
- ⚛️ React/Vue 组件 - 快速生成组件代码
- 📐 正则表达式 - 根据需求生成 Regex
- ☁️ AWS 基础设施 - Terraform 配置
